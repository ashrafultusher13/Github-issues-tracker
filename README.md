1️⃣ What is the difference between var, let, and const?
Ans. 
Var is the older way to declare variables. It is function-scoped and it can be both redeclared and updated.
Let is the new way to declare variables. It is block-scoped and it can be updated but cannot be redeclared.
Const is also block-scoped like let, but the difference is that once a value is assigned to a const variable, it cannot be updated or redeclared.


2️⃣ What is the spread operator (...)?
Ans.
The spread operator (...) is used to expand elements of an array or properties of an object.
It is used to copy arrays or objects, merge them, add new elements without modifying the original data.
Example: We can use the spread operator to combine two arrays or create a new array from an existing one.

3️⃣ What is the difference between map(), filter(), and forEach()?
Ans.
map() goes through each element of an array and returns a new array after modifying each element.
filter() also returns a new array, but it only includes the elements that satisfy a specific condition.
forEach() is used to loop through the array and perform an action for each element, but it does not return a new array.

5️⃣ What are template literals?
Ans.
Template literals are a way to create strings in JavaScript using backticks (`) instead of single or double quotes.
They allow us to easily insert variables inside a string using ${} syntax.
Template literals also support multi-line strings, which makes writing long text easier.
